# EFinance

Projeto de um sistema de gerenciamento de Finanças pessoal.
V1.0

## Instalação

Para instalar as dependências, use o seguinte comando:

```bash
npm install
```

## Versão do Node

Este projeto foi desenvolvido com Node.js 

## Como Rodar

Para rodar o projeto, use o seguinte comando:

```bash
npm start
```
